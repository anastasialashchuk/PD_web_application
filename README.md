# PD_web_application
Приложение к ВКР "Разработка аппаратно-программного комплекса для оценки параметров двигательной активности рук по 2D-видео".

Для разработки веб-сервиса был выбран следующий технологический стек: Flask, SQLAlchemy, Blueprint – для бэкенда, Jinja2, Bootstrap и HTML с CSS – для фронтенда. Веб-приложение написано на языке Python с использованием ключевой библиотеки MediaPipe для детекции контрольных точек руки.
